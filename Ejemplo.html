HTML

HTML son las siglas de Hypertext Markup Language que puede traducirse como leguaje de marcas o lenguaje de hipertexto.

El lenguaje HTML se emplea para crear páginas web. Es muy fácil ver el código HTML de una página web la opción exacta cambia de un navegador a otro y también puede cambiar de una versión a otra de un navegador, pero suelen tener un nombre similar.

HTML se componen de etiquetas que se escriben entre los simbolos mayor que y menor que.

Historia de HTML

Los inicios del lenguaje HTML se remontan al año 1990, cuando Tim Bemeers-Lee creó la primerapágina web

Versiones de HTML

Tim Bemeers-Lee definió la primera versión de HTML en el año 1991.

En la actualidad la versión más actualizada de Versiones de HTML es HTML 5.
